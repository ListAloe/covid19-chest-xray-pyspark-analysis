# Система анализа медицинских изображений для эпидемиологического мониторинга COVID-19

## Описание проекта
Аналитическая система для мониторинга COVID-19 на основе метаданных рентгеновских снимков с использованием PySpark.

## Требования
- Python 3.7+
- PySpark 3.0+
- Google Colab (рекомендуется)

## Запуск проекта
1. Откройте Google Colab
2. Загрузите файл `analysis.ipynb`
3. Запустите все ячейки последовательно
4. Результаты сохранятся в формате Parquet

## Структура проекта
- Подготовка среды PySpark
- Загрузка и предобработка данных
- Анализ качества данных
- SQL-аналитика (5 запросов)
- Обработка данных в PySpark
- Визуализация результатов
- Выводы и рекомендации
